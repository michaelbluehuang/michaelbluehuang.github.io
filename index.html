LeetCode周赛日

Posted on 2020-04-12

今天完成了LeetCode周赛初体验，感觉这个活动非常棒，主要体现在：

  • 能够受到排名和奖励的机制的激励，增加刷题的动力
  • 竞赛页面的设计保持了LeetCode一贯的简洁、优雅的风格，让刷题变成一种享受
  • 当然最重要的还是通过规定时间内的训练能够提高用户编码和算法设计的能力

本次题目有四，总体难度不高：

  1. 数组中字符串的匹配 这道题大概只能两层循环暴力求解，主要考查的是用字符串处理中的find()函数来判断子串，当存在子串关系是返回的是子串首字母在母串对应的下标，否则返回std::string::npos（是一个特定的整数）
  2. 查询带键的排列 题目叙述很冗长，但是本质就是数组元素的查找和移动，考察基本功
  3. HTML实体解析器 写代码的时候有些麻烦，要对每一个特殊字符进行一个条件判断，考察的是字符串中子串位置的判断，可以使用substr()函数解决
  4. 给N x 3网格图涂色的方案数 动态规划，利用数学思维得到网格图行与行之间的递推关系，由于最终结果的数字会非常大，dp数组采用long类型

前些天偶然在周赛排行榜上发现一位大神，他在每次周赛的时候会将自己的写码过程录下来并传到B站上，每次比赛全部题目ac的时间经常在15分钟左右······通过观看他的解题过程，才终于认识到那些信息学竞赛选手的思维速度和编码能力处于怎样一种常人无法企及的高度。但他们就是有别于常人的天才吗？我认为绝大多数不是，无数个日夜的刻苦练习和积累才是“我们”与他们之间最大的不同。

源码、反码、补码总结

Posted on 2020-03-29

三者为机器存储一个具体数字的编码方式

原码

第一位表示符号，其余位表示真值的绝对值：

[+1]原 = 0000 0001

[-1]原 = 1000 0001

取值范围：[1111 1111 , 0111 1111] 即 [-127 , 127]

反码

正数的反码是其本身，负数的反码是在其原码的基础上, 符号位不变，其余各个位取反：

补码

正数的补码是其本身，负数的补码是在其原码的基础上, 符号位不变, 其余各位取反, 最后+1：

[+1] = [00000001]原 = [00000001]反 = [00000001]补

[-1] = [10000001]原 = [11111110]反 = [11111111]补

位运算的应用

Posted on 2020-03-28
  • 计算机进行位运算的速度要快于加减运算。
  • ”数组中数字出现的次数“系列问题用位运算来解决非常方便和高效

异或

首先，异或运算有如下重要的性质：

  • 交换律
  • 结合律
  • a ^ a = 0
  • a ^ 0 = a

在掌握了以上性质之后，可以得到如下结论：

当数组中有一个元素出现一次，其余元素出现两次时，对所有元素进行异或运算的结果便是只出现一次的元素，即：

b ^ a ^ c ^ b ^ c = a

不要再迷失了！！

Posted on 2020-03-25

距离春招完全结束，仅剩不到一个月的时间了。

突然发现自己经常用来激励自己的鸡汤配方往往只能在短时间内卓有成效，过了这个劲儿，整个人又会重新陷入那个沉缓、毫无生气的节奏。

其实一个人有信仰是一件非常伟大的事情，因为在最困难的时候，他明白他所坚信的一切会带他走出困境，而不是不知所措，坐以待毙。

做事有原则，有一套自己的方法论也很了不起，面对问题、解决问题时的勇气往往就来源于此。

这让我想到了在职业篮球比赛当中，在比赛的关键时刻每支球队所采用的，一定是其最本源、最有把握的战术打法。这时，每个球员都回到了自己最舒服的位置上，做着自己最擅长的事，而过往那一场场艰难而伟大的胜利在脑海中浮现时，他们的自信完全被点燃，他们目光如炬，因为这就是他们的信仰，他们的方法论。

虽然在家这段时间效率很低，但是自己也做了一些反思，尝试从曾经那些惨痛的教训中总结出一些能使未来的自己受用的方法。希望在以后的生活中，自己能够：

  • 提高学习效率（必要的时候使用番茄钟）
  • 保持专注，心中时刻要有不能为外界所扰的明确的目标
  • 提高思维活跃度，加快反应速度
  • 经常和自己对话，坚持反思和总结，不忘初心
  • 找到自己的兴趣爱好，并坚持下去

放手一搏

Posted on 2020-03-12

已是三月中旬，我仍然呆在家里——受新冠肺炎的影响，2020年的这个春天迟迟未到。

每个人都在经受着一场严峻的、深刻的考验。对于我来说，本应该相对轻松一些的大四下学期被我活生生改造成了一场残酷的战斗：我不仅要面对繁琐的毕设，还要为职业生涯（出国）做准备。而我只有一个选择，那就是赢下去。我要战胜的是我自己，那个曾经不知道努力的自己，那个知道努力了但是效率很低、不会思考的自己，那个会计划、会思考了却又一再拖延、执行力很差的自己。

在回学校之前，我要完成的任务有：

  • 找到一份远程实习
  • 毕设任务的基本实现
  • 找到一份gap year的工作
  • 熟记GRE单词

不是你不够勇敢，而是你没有发现那个勇敢的自己。带着梦想和责任勇敢地去拼搏吧，赋予时光应有的价值，给青春和人生增添应有的重量。
